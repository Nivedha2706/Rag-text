## 🚀 Features

✨ Multi-format Document Support
🧠 Embeddings with `nomic-embed-text`
💾 ChromaDB Vector Storage
🔍 Semantic Retrieval
🤖 Llama 3.2 : 3B Integration
🌐 Streamlit Interface
📚 Detailed Summarization
💬 Question Answering

---

## ⚙️ Architecture

```mermaid
graph TD
A[📄 User Document] --> B[✂️ Chunking]
B --> C[🧠 Embeddings]
C --> D[💾 ChromaDB]
D --> E[🔍 Retriever]
E --> F[🤖 Llama 3.2]
F --> G[💬 Intelligent Answer]
```
